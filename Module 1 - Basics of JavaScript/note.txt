1️⃣ Introduction to JavaScript
# Explanation:
JavaScript is a programming language that runs in browsers (and on servers via Node.js). It makes web pages interactive, like showing alerts, updating content dynamically, validating forms, and more.

# Key Points:
- Developed in 1995 by Brendan Eich.
- Works with HTML & CSS (HTML = structure, CSS = style, JS = behavior).
- Can run in browsers or on servers.

2️⃣ Setting Up Environment
# Explanation:
You can write JS in:
- Browser Console: Press F12 → Console tab.
- Inline JS: <script>alert('Hello');</script>
- Internal JS: <script>console.log('Hi');</script> inside HTML <head> or <body>
- External JS: Create script.js and link it:
<script src="script.js"></script>

3️⃣ First JavaScript Program
console.log("Hello, World!");

- console.log() → outputs to developer console.
- alert() → shows a pop-up to the user.

4️⃣ Variables & Constants
Explanation:
- var → old way, function-scoped, can be re-declared.
- let → block-scoped, can be updated but not re-declared in same scope.
- const → block-scoped, cannot be updated or re-declared.

let age = 25;  // can change
const name = "Mehedi";  // cannot change
var city = "Doha";  // function scoped

5️⃣ Data Types
- Primitive Types: string, number, boolean, undefined, null, symbol, bigint
- Reference Types: object, array, function

let name = "Mehedi"; // string
let age = 25; // number
let isStudent = true; // boolean
let person = {name: "Mehedi", age: 25}; // object
let colors = ["red", "green", "blue"]; // array

6️⃣ Operators

# Types & Examples:
Arithmetic: + - * / % **
let x = 10 + 5; // 15

Assignment: =, +=, -=
let y = 10; y += 5; // 15

Comparison: ==, ===, !=, !==, >, <
console.log(5 == "5"); // true (value only)
console.log(5 === "5"); // false (value & type)

Logical: &&, ||, !
console.log(true && false); // false

Ternary: condition ? true : false
let age = 18;
let canVote = (age >= 18) ? "Yes" : "No"; // Yes
