1. Search Bar with Auto-Suggestions
Example: Google search bar or e-commerce websites like Amazon.
JavaScript use: JS listens for what you type and dynamically shows suggestions without reloading the page. It can also fetch suggestions from a server using AJAX or fetch API.

2. Image Carousel/Slider
Example: Homepage banners on news websites or online stores.
JavaScript use: JS changes the images at timed intervals, responds to arrow clicks, and allows swiping on mobile devices.

3. Form Validation
Example: Login forms or signup pages on social media sites.
JavaScript use: JS checks if inputs are valid (like email format, password strength) before sending data to the server, giving instant feedback to users.
